Features

Glassmorphism

Contemporary design concept that combines transparency and blur effects to create an elegant, frosted glass-like appearance in web interfaces. It provides a polished look to UI elements.

Box Shadow

Experience to easily add depth by creating subtle shadows around elements, contributing to a sense of dimensionality and hierarchy in web design.

Gradiefy

Powerful design tool used in web interfaces creating a gradual transition between two or more colors, adding depth and interest to elements like buttons, backgrounds, and logos.
